# Käyttöohje


## Ohjelman käynnistäminen

Ohjelma käynnistetään komennolla
```
gradle run
```

## Lukuvinkin luominen

Uusi linkki tallennetaan komennolla 
```
T
```


## Lukuvinkkien listaaminen

Linkit listataan komennolla
```
L
```


## Sovelluksen sulkeminen

Sovellus suljetaan komennolla
```
Q
```
